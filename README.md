## 💼 Featured Projects

### 🚕 [NYC Taxi Trip Duration Prediction — End-to-End ML Pipeline & Deployment](https://nyc-taxi-trip-duration-api.onrender.com/docs)
`Python` `Pandas` `NumPy` `Scikit-learn` `Random Forest` `FastAPI` `Docker` `Render`

- Built an end-to-end ML pipeline on **1.45M+ NYC taxi trip records** — cleaned invalid coordinates/outliers, ran EDA, and engineered time-based, peak-hour, and Haversine distance features (distance emerged as the top predictor at **≈82.75%** importance).
- Trained and benchmarked Linear Regression, Decision Tree, and Random Forest models; applied log transformation (`log1p`/`expm1`) to correct skew, boosting the final Random Forest to **R² = 0.7674** (MAE 191.11s, RMSE 318.94s).
- Shipped as a **production REST API** with FastAPI + interactive Swagger docs, containerized with Docker, versioned via Git/GitHub (Git LFS for model artifacts), and **deployed live on Render**.
- 🔗 **Live Demo:** [nyc-taxi-trip-duration-api.onrender.com/docs](https://nyc-taxi-trip-duration-api.onrender.com/docs)

### 🚨 District-wise Crime Pattern Analysis — ML & Statistical Modelling
`Python` `Pandas` `NumPy` `Scikit-learn` `Seaborn` `Matplotlib`

- Engineered an end-to-end ML pipeline with ETL, EDA, and correlation-based feature selection over **800+ district-level NCRB crime records**, reducing 20+ raw features to the most statistically significant predictors.
- Trained and cross-validated a Linear Regression model (**R² = 0.96** on held-out data); applied residual analysis and compared alternate models to confirm generalization and rule out overfitting.
- Designed statistical visualizations (heatmaps, regression plots, distribution charts) and published a full methodology report with codebase, earning **16+ GitHub stars** from the open-source community.

---

### 📊 Crime Analytics Dashboard — India (National Scale)
`Power BI` `DAX` `Data Normalization` `Statistical Analysis` `ETL`

- Architected a multi-level Power BI dashboard over national crime datasets, engineering a custom **Risk Index** (crimes per 100,000 women) via DAX to normalize raw event counts into a statistically comparable per-capita metric.
- Implemented drill-down navigation across three geographic hierarchies (national → state → district) with interactive slicers, reducing manual multi-table analysis to **sub-second filtered views**.
- Applied per-capita normalization to surface non-obvious risk patterns — demonstrating that high raw totals do not correlate with high per-capita risk, a finding invisible to raw-count analysis.
